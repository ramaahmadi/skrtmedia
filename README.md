# SKRT Media

Website media untuk SKRT (Satuan Komunitas Rayon Tiga)

## Deskripsi

Website ini dibangun menggunakan Next.js dengan fitur-fitur:
- Manajemen kegiatan dan acara
- Sistem registrasi online
- Gallery foto dan dokumentasi
- Informasi kontak dan lokasi
